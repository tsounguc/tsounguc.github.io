Overview

Machine learning (ML), an application of AI, is a programming technique that allows apps to learn and improve from experience without being explicitly programmed to do so. This usually goes well with apps that have unstructured data such as images and text, or problems with large number of parameters such as predicting the winning sports team. Some uses cases for machine learning include:

  • Image labeling: detecting various objects in a given image. The ML models are trained to identify the objects.
  • Optical Character Recognition (OCR): Detecting a written text and inferring its script
  • Barcode Scanning: Read data encoded using most standard barcode formats
  • Smart Reply: automatically generate relevant replies to messages. Smart Reply helps your users respond to messages quickly, and makes it easier to reply to messages on devices with limited input capabilities.

Getting started.

Before we dive into the coding part, let’s see what is needed to complete this tutorial.

IDE version

We will be using Android Studio for our IDE and will be using Java as the language. As you can see the current IDE version we’re using is 3.5.2 any new version after that will be acceptable. To know your android studio version go to File > Settings ... (Settings dialog appears) ... Appearance & Behaviour > System Settings > Updates. Here, both the current version and the build number are shown.

Dependencies

Let’s add required libraries in build.gradle and permissions in Manifest.

In build.gradle (Module: app) depencies add the following line

//image to text google library

implementation 'com.google.android.gms:play-services-vision:16.2.0'

Next go to manifests/AndroidManifest.xml and add the following lines as shown in the picture below

<meta-data android:name="com.google.android.gms.vision.DEPENDENCIES"

   android:value="ocr"></meta-data>

Once you’ve entered all dependencies, click Sync Now
